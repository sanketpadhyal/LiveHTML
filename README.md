## 🚀 Features  

- 👨‍💻 **Live HTML/CSS/JS rendering** in the browser  
- 💡 **Smooth transitions** with glowing UI effects  
- 🖥 **Responsive layout** for PC and 📱 mobile devices  
- 🔥 Built with **pure HTML, CSS, and JavaScript** (no frameworks)
